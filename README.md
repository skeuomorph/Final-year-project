# Final-year-project
Causal inference of Stop and Search data

## Data gathered
See "data" folder for data gathered for this project. Taken from the open-source data.police.uk website.

## disparity.ipnyb
A Jupyter notebook containing aggregates of data depending on ethnicity, age, and gender. Matplotlib used for graphs.

## KL_divergence.ipnyb
A Jupyter notebook which explores the Kullback-Leibler divergence between the volume of Stop and Search, and Crimes Reported. This is a means of exploring the claim that "Stop and Search prevents crime".
